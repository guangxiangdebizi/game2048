## 💡 实用技巧

### 如何在 Windows 上截图手机屏幕？

**方法一：使用 Android Studio 模拟器**
1. 打开 Android Studio
2. 启动模拟器 (AVD Manager)
3. 安装您的 APK
4. 玩游戏并截图（模拟器工具栏有截图按钮）

**方法二：使用 ADB + scrcpy**
1. 安装 scrcpy：https://github.com/Genymobile/scrcpy
2. 连接手机到电脑
3. 运行 scrcpy，手机屏幕会镜像到电脑
4. 用电脑截图工具截图

**方法三：直接用手机截图**
1. 在手机上截图
2. 通过微信/QQ 传到电脑
3. 简单快捷！

### 如何生成多尺寸图标？

**推荐工具**：
1. **Icon Kitchen** (在线，免费)
   - https://icon.kitchen/
   - 上传 `icon.png`
   - 自动生成所有尺寸

2. **Android Asset Studio** (在线，免费)
   - https://romannurik.github.io/AndroidAssetStudio/
   - 功能更丰富

3. **Figma/Photoshop** (本地)
   - 手动调整大小
   - 导出为 PNG

### 应用商店审核被拒怎么办？

常见拒绝原因：
1. **图标/截图不符合要求** → 检查尺寸和格式
2. **隐私政策缺失** → 使用 `PRIVACY_POLICY.md`
3. **应用描述不清晰** → 使用 `STORE_LISTING.md`
4. **应用崩溃** → 在多个设备上测试
